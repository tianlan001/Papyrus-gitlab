This folder contains models built using Papyrus and its extension for the manufacturing domain.
The smart mold models available in this folder has been developed in the frame of the H2020 project BEinCPPS http://beincpps.eu

To open the smart models, you need to install :
1. Eclipse Neon 
2. Papyrus UML from this update site: http://download.eclipse.org/modeling/mdt/papyrus/updates/releases/neon
3. Papyrus SysML from this update site: http://download.eclipse.org/modeling/mdt/papyrus/updates/releases/neon
4. BPMN feature from this update site: http://download.eclipse.org/modeling/mdt/papyrus/updates/releases/neon
5. BMM feature from this update site: http://download.eclipse.org/modeling/mdt/papyrus/updates/releases/neon
6. Papyrus4Manufacturing feature from this update site: http://download.eclipse.org/modeling/mdt/papyrus/updates/releases/neon
